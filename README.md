# isthisflag-backend

Backend for the "Is This Flag?" turn-based flag guessing game.

## Stack
- Node.js
- Express
- Socket.IO

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the server:
   ```sh
   npm start
   ```
3. Health check endpoint:
   - GET http://localhost:3001/health
